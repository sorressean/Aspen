Detailed Description

The FactoryManager is responsible for managing individual factories used to create objects. This allows us to create and/or modify specific objects, as long as they conform to a specific interface from extensions. For example, if you wanted to add combat to a player you might have functions like DoAttack, etc. These functions shouldn't be added to the engine's player object, so the extension system would expose a factory. You would register your new player object with the factory, and any time a player was created, it would be requested from the factory.
