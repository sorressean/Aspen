Detailed Description

Contains: Component

The component class is responsible for adding functionality to an object. With components, objects are no longer has-a relatoinships, but has-a relationships. For example, in order to enable weapons to be worn (like gloves with claws), weapon logic would need to be part of the inheritance structure or enable all objects to be worn. This is problematic, because not every object should be a weapon or worn in certain scenarios. Components enable you to add the functionality to individual objects without complicating the inheritance structure of the object. Components enable for the extension of objects.
